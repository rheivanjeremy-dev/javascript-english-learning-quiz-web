 📚 Direct & Indirect Speech Learning Website

An interactive educational website designed to help students understand and practice Direct & Indirect Speech in English. This project combines learning materials, grammar explanations, examples, and interactive quizzes in a modern and user-friendly interface.

 ✨ Features
* 📖 Complete learning materials about Direct & Indirect Speech
* 💬 Explanation of Direct Speech and Indirect Speech
* 🔄 Backshift Tenses reference table
* 📅 Time Expression conversion table
* 📝 Interactive quiz with 20 multiple-choice questions
* ✅ Instant answer feedback
* 📊 Automatic score calculation
* 🔄 Quiz reset functionality
* 🎨 Modern dark-themed responsive design

 🛠 Technologies Used
* HTML
* CSS
* JavaScript 

 🎯 Learning Objectives
This website was developed to help students:

* Understand the differences between Direct and Indirect Speech
* Learn tense changes (Backshift)
* Practice converting direct speech into indirect speech
* Improve English grammar skills through interactive exercises
